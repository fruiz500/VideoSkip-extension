VideoSkip Ajuda

Esta extensão permite que você filtre vários tipos de conteúdo questionável de streams de vídeo. Ele usa as mesmas categorias da seção "Parents Guide" do site IMDB.com. Ele pode, ao seu critério, pular seções inteiramente, ou simplesmente silenciar o som ou apagar o vídeo. Para começar a usar o VideoSkip, você precisa de somente duas coisas:

  1. O filme que você deseja editar enquanto ele é transmitido. Vá para o seu serviço de streaming favorito e alugue o filme para que esteja pronto para ser reproduzido (sem anúncios, na Amazon). O filme deve ser iniciado antes de você clicar no botão VideoSkip.
  2. Um arquivo "skip" que informa ao aplicativo o que ignorar e quando, com extensão .skp. Uma boa fonte é o VideoSkip Exchange (há um botão para isso), mas existem outras fontes.

Short instructions:

  1. Com o filme pronto para ser reproduzido em seu navegador, clique no botão VideoSkip no canto superior direito do navegador. A interface do VideoSkip aparecerá ao lado do vídeo.
  2. Carregue um arquivo skip para este filme usando o botão Carregar arquivo skip.
  3. Se o arquivo skip conhecer sua fonte de streaming, você será direcionado para a guia de nome Filtro . Defina os filtros de categoria como desejar. Quanto mais alto, mais rigoroso é o filtro. Uma descrição detalhada dos níveis de filtragem aparece abaixo dos controles deslizantes conforme você os move.
    • Se a fonte de streaming for nova ao arquivo skip, você será direcionado para a guia de nome Sincronizar primeiro, para que possa sincronizar os avanços com sua fonte de streaming. Abaixo, uma descrição detalhada do processo:
  4. Oculte a interface do VideoSkip clicando em seu logotipo ou no X no canto superior esquerdo e continue reproduzindo o filme normalmente. Os avanços ocorrerão no devido momento. Se quiser vê-lo em uma tela maior, conecte-o ao computador com um cabo; não use um screen cast.

Clique nos títulos abaixo para ler instruções mais detalhadas.

Como filtar um vídeo

  1. Carregue o vídeo de um serviço de streaming normalmente e comece a reproduzir seu vídeo, então pare e clique no ícone VideoSkip no canto superior direito do navegador. Uma nova janela aparecerá ao lado do vídeo. Você pode movê-lo ou redimensioná-lo como uma página normal com Ctrl + ou Ctrl -.
  2. Se você tiver um arquivo .skp contendo os itens a serem avançados, poderá carregá-lo agora com o botão Carregar arquivo skip na guia Carregar. Depois de carregado, seu conteúdo aparecerá em uma caixa na guia Editar.
    • Se você não tem um arquivo skip para o filme que está assistindo, é provável que outro usuário tenha feito um e postado no VideoSkip Exchange. Há um botão para carregar o Exchange em uma guia separada. Você não precisa se registrar para navegar e fazer o download de arquivos skips.
  3. Se o arquivo skips tiver sido sincronizado anteriormente com seu serviço de vídeo, você será direcionado para a guia Filtro para que possa definir as intensidades do filtro conforme descrito na próxima etapa. Caso contrário, você será direcionado para a guia Sincro. Se a imagem corresponder à imagem na guia, você está sincronizado e pode começar a assistir ao filme. Se isso não acontecer, entretanto, você terá que esfregar o vídeo com as teclas de seta que aparecerão, conforme explicado na seção abaixo, até que isso aconteça. Você pode querer sobrepor a captura de tela com o botão Sobrepor. Quando a captura de tela finalmente corresponder ao vídeo, clique em Sincronizar tempos para sincronizar tudo e remover a captura de tela sobreposta. Uma nova versão do arquivo pular, que será sincronizado automaticamente com seu serviço de vídeo, será criada na pasta Downloads para que você possa compartilhá-lo com outras pessoas.
  4. Defina os filtros de categoria usando os controles deslizantes na guia Filtro, de acordo com o público. Quanto mais alto, mais rigoroso. Alto = para crianças, Médio = para adolescentes, Baixo = para adultos. Se um filtro estiver desativado, o conteúdo sinalizado será mostrado. Depois disso, você está pronto para assistir.
    • VideoSkip inclui um filtro automático de palavrões, que é ativado clicando no botão próximo à parte inferior desta guia. Você será solicitado a carregar um arquivo de legenda para o vídeo, que será verificado em busca de palavras na lista de bloqueio editável (clique na descrição para exibi-la ou editá-la). Esta ação não exibirá legendas para o vídeo, que deve ser carregado do serviço de streaming. O tempo geral desses avanços pode ser ajustado com os botões de seta, após checar Alt.
  5. Se você quiser alterar algo na lista de avanços, vá para a guia Editar. Há um botão para inserir a hora atual no vídeo (o próprio vídeo tem uma barra de rolagem para acessá-la) e uma " seta " formatada corretamente se você estiver com preguiça. Há também um botão para salvar sua lista de pular editada na pasta padrão para downloads. Digitar duas barras // na segunda linha de qualquer salto desativa isso.
  6. Oculte a janela VideoSkip, inicie o vídeo em sua própria página, relaxe e divirta-se. Os avanços ocorrerão quando chegar a hora deles. Uma mensagem aparecerá no vídeo em tela cheia quando você mover o mouse, informando quais filtros estão ativados.

Como sincronizar os avanços

  1. Se o vídeo corresponder à captura de tela na guia Sincronizar depois de carregar o arquivo de pular, tudo está sincronizado e essa fase está concluída. Deslize o vídeo de volta ao início e comece a assistir.
  2. Mas se o vídeo for de uma fonte diferente daquela usada para fazer o arquivo de sincronização, poderá haver uma incompatibilidade. Use os botões de seta para mover o vídeo até que a tela corresponda à captura de tela ou descrição (quadro a quadro, se Fino estiver marcado). O botão de avanço rápido alterna a velocidade ou retoma a reprodução se estiver em pausa. Você pode mover a captura de tela com o mouse. Ele será redimensionado se você pressionar a tecla Alt ou se tiver Alt checado.
  3. Nos serviços que permitem capturas de tela, o VideoSkip possui um processo automático para encontrar o tempo da captura de tela clicando no botão Encontrar foto, para serviços que permitem fazer capturas de tela. Certifique-se de que você está dentro de um segundo do tempo correto antes de clicar nele.
  4. Clique no botão Sincronizar tempos quando o vídeo e a captura de tela coincidirem. Isso deslocará todos os avanços na medida certa, de modo que acontecerão nos momentos corretos. Uma nova versão do arquivo ignorado será criada para que as pessoas com quem você o compartilhe não tenham que repetir o processo de sincronização.

Criando seus próprios avanços

  • Um arquivo skip bem produzido terá uma primeira entrada com o tempo de um evento próximo ao início do vídeo, onde a imagem muda rapidamente, além de uma descrição na linha abaixo dela (não use números na descrição), e uma captura de tela. Isso permite que os visualizadores usem os avanços no arquivo, mesmo se estiverem assistindo a um vídeo obtido de uma fonte diferente, que pode ter um tempo diferente. Para fazer uma captura de tela, basta deslizar o vídeo até o tempo desejado e clicar em Fazer captura de tela; a hora será adicionada à posição atual do cursor. Se o serviço não permitir capturas de tela, aparecerá um botão para que você possa carregá-lo manualmente. Nesse caso, faça a captura de tela com o método integrado ao sistema operacional (" ferramenta de recorte " no Windows, cmd-shit-5 no MacOS), salve-o em um arquivo e carregue o arquivo no VideoSkip usando o novo botão. Se a tela contiver áreas pretas, é melhor excluí-las e capturar apenas o filme adequado.
  • Cada avanço consiste nos tempos de início e término em relação ao início do vídeo, com uma seta entre eles e na linha abaixo de uma categoria mais talvez um rótulo de manuseio (como " vídeo " ou " audio "). Inserir pulos é tão fácil quanto arratas até o ponto onde pular / silenciar / em branco deve começar, clicando no botão Inserir hora em uma nova linha e, em seguida, na Inserir seta, deslizando até o ponto em que o salto deve terminar, clicando no botão Inserir hora mais uma vez e, em seguida, indo para a próxima linha e escrevendo um conteúdo, classificação numérica e etiqueta de manuseio (opcional). Se você não tiver certeza de onde os itens omitidos devem ser inseridos, há um " Avanço rápido " botão que limpa o vídeo em alta velocidade. Clique novamente para parar. O botão à esquerda também interrompe o vídeo. Há também um botão Silêncio que insere um silenciamento retroativo de uma única palavra para profanação; clique nele assim que ouvir a palavra enquanto o vídeo é reproduzido normalmente.
  • Os rótulos de conteúdo não diferenciam maiúsculas de minúsculas. Eles consistem em qualquer palavra em inglês (na verdade, as três primeiras letras são suficientes) das categorias IMDB, que são exibidas ao lado das caixas de seleção, como "sex", " nudity", "gore", "drugs", Rótulos de tratamento são as seguintes palavras: "audio", "sound", "video", "image", "blur", "fast" (ou as três primeiras letras de cada, incluindo "img") também não faz distinção entre maiúsculas e minúsculas.
  • O " audio " ou "sound" palavras-chave farão com que o som seja silenciado durante o intervalo determinado enquanto o vídeo ainda é mostrado, em vez de avançar a seção completamente, o que pode ser útil para remover palavrões. O "video" e "image" palavras-chave fazem com que a imagem fique em branco enquanto o som ainda é reproduzido, o que pode ser útil em casos de nudez, etc. "blur" faz com que a imagem fique desfocada. "fast" irá passar o filme em avanço rápido em vez de pular para o final do intervalo marcado. Nenhuma palavra-chave de manipulação significa que a seção será completamente avançada.
  • As palavras-chave "blank" e "blur" podem afetar apenas uma pequena parte da tela, se desejado. Para fazer isso, clique no botão "Turva" para fazer aparecer um quadrado embaçado no vídeo. Mova-o com o mouse (redimensione-o segurando também a tecla Alt). Quando estiver satisfeito, clique no botão "Pos. turva" para inserir a posição exata da caixa de desfocagem no salto, como uma lista de quatro números entre parênteses rectos. Tenha em mente que na reprodução a caixa de desfocagem não se moverá durante o tempo desse salto.
  • Por fim, adicionar um número de 1 (leve) a 3 (grave) permitirá que os usuários definam o nível de filtragem de forma adequada. Um bom padrão é o seguinte: 1 = problemático para crianças, 2 = problemático para adolescentes, 3 = problemático para adultos. Há uma rúbrica detalhada sobre o que isso pode significar no Videokip Exchange. Você pode adicionar texto explicativo entre parênteses, o que não acionará os filtros.
  • Tudo é editável na caixa. Use-o para ajustar os tempos. Por exemplo, palavras para serem silenciadas normalmente levam muito menos de um segundo. Você pode revisar cada tempo selecionando na lista e clicando em Ir para o tempo. Se a hora for selecionada, as setas mudarão a hora e também limparão o vídeo. O botão " Inserir " os botões colocam as coisas sempre na posição atual do cursor.
  • Certifique-se de que haja uma linha em branco entre os saltos, compondo uma linha com os tempos e uma linha com rótulos abaixo dela.
  • O botão Salvar salvará sua lista, além da captura de tela, no formato .skp em sua pasta de downloads padrão. Você terá que movê-lo de lá para a pasta onde deseja. Este formato é composto por texto, então você pode visualizar o arquivo em qualquer editor de texto.
  • O botão VideoSkip Exchange na guia Carregar abrirá uma página da web onde você pode compartilhar seu arquivo skip com outros usuários do VideoSkip.
