Aide VideoSkip

Cette extension vous permet de filtrer plusieurs types de contenu répréhensible de vidéo. Il utilise les mêmes catégories que le "Guide des parents" de IMDB.com. Il peut sauter des sections entièrement, ou simplement couper le son ou vider la vidéo, à votre discrétion. Pour commencer, vous avez besoin de deux choses:

  1. Un film que vous souhaitez modifier au fur et à mesure de sa diffusion. Veuillez vous rendre sur votre service de streaming préféré et louer le film pour qu'il soit prêt à être lu (sans aucune publicité, sur Amazon). Le film doit s’afficher avant que vous ne cliquiez sur le bouton VideoSkip.
  2. Un fichier qui indique à l'application quoi filtrer et quand, avec l'extension .skp. Une bonne source est VideoSkip Exchange (il y a un bouton pour cela), mais il existe d'autres sources.

Brèves instructions:

  1. Une fois le film prêt à être lu sur votre navigateur, cliquez sur le bouton VideoSkip dans le coin supérieur droit du navigateur. L'interface VideoSkip apparaîtra à côté de la vidéo.
  2. Chargez un fichier à filtrer pour ce film à l'aide du bouton Charger fichier .skp.
  3. Si le fichier filtré connaît votre site de streaming, vous serez redirigé vers l'onglet Filtre. Définissez les filtres de catégorie de manière appropriée. Plus le filtre est élevé, plus le filtre est rigoureux. Une description détaillée des niveaux de filtrage apparaît sous les curseurs lorsque vous les déplacez.
    • Si le site de streaming est nouveau pour le fichier de filtre, vous serez d'abord dirigé vers l'onglet Sync afin que vous puissiez synchroniser les sauts avec votre source de diffusion. Vous trouverez ci-dessous une description détaillée du processus.
  4. Masquer l'interface VideoSkip en cliquant sur son logo ou sur le X dans le coin supérieur gauche et continuer à lire le film normalement. Les sauts auront lieu en temps voulu. Si vous voulez le voir sur un écran plus grand, connectez-le à l'ordinateur avec un câble; n'utilisez pas le partage d'écran.

Cliquez sur les titres ci-dessous pour lire des instructions plus détaillées.

Comment filtrer une vidéo

  1. Chargez la vidéo à partir d'un service de streaming normalement et commencez la lecture afin qu'elle charge les données, puis arrêtez-la et cliquez sur l'icône VideoSkip en haut à droite du navigateur. Une nouvelle fenêtre apparaîtra à côté de la vidéo. Vous pouvez la déplacer ou la redimensionner comme une page normale avec Ctrl + ou Ctrl -.
  2. Si vous avez un fichier .skp contenant les filtres, vous pouvez le charger maintenant avec le bouton Charger fichier .skp dans l'onglet Charger. Une fois chargé, son contenu apparaîtra dans une boîte sur l'onglet Modifier.
    • Si vous n'avez pas de fichier de saut pour le film que vous regardez, il est probable qu'un autre utilisateur en ait créé un et l'ait publié dans le VideoSkip Exchange. Il y a un bouton pour charger l'échange sur un onglet séparé. Vous n'avez pas besoin de vous inscrire pour parcourir et télécharger les fichiers ignorés.
  3. Si le fichier ignoré a déjà été synchronisé avec votre service vidéo, vous serez redirigé vers l'onglet Filtre afin que vous puissiez définir les niveaux de filtrage comme décrit à l'étape suivante. Sinon, vous serez redirigé vers l'onglet Sync. Si l'image correspond à l'image sur l'onglet, vous êtes synchronisé et pouvez commencer à regarder le film. Si ce n'est pas le cas, vous devrez alors faire défiler la vidéo avec les touches fléchées qui apparaîtront, comme expliqué dans la section ci-dessous, jusqu'à ce que ce soit le cas. Vous pouvez superposer la capture d'écran avec le bouton Superposer. Lorsque la capture d'écran correspond enfin à la vidéo, cliquez sur Synchroniser les heures pour tout synchroniser et supprimer la capture d'écran superposée. Une nouvelle version du fichier de filtre, qui se synchronisera automatiquement avec votre service vidéo, sera créée dans votre dossier Téléchargements afin que vous puissiez le partager avec d'autres.
  4. Définissez les filtres de catégorie à l'aide des curseurs de l'onglet Filtre, en fonction de l'audience. Le plus élevé, le plus strict. Élevé = pour les enfants, Moyen = pour les adolescents, Faible = pour les adultes. Si un filtre est désactivé, le contenu signalé sera affiché. Après cela, vous êtes prêt à regarder.
    • VideoSkip inclut un filtre automatique de jurons, qui est activé en cliquant sur le bouton situé en bas de cet onglet. Il vous sera demandé de charger un fichier de sous-titres pour la vidéo, qui sera analysé à la recherche de mots sur la liste de blocage modifiable (cliquez sur la description pour l'afficher ou la modifier). Cette action n'affichera pas les sous-titres de la vidéo, qui doivent être chargés à partir du service de streaming. Le timing global de ces sauts peut être ajusté avec les flèches, après avoir enfoncé Alt.
  5. Si vous souhaitez modifier quelque chose dans la liste des sauts, accédez à l'onglet Modifier. Il y a un bouton pour calibrer le bon moment sur la vidéo (la vidéo elle-même a une barre de défilement pour y accéder), et même une "flèche" si vous vous sentez paresseux. Il existe également un bouton pour enregistrer votre liste de sauts modifiée dans le dossier par défaut pour les téléchargements. Tapez deux slashs // dans la deuxième ligne de tout saut pour le désactiver.
  6. Masquer la fenêtre VideoSkip, démarrer la vidéo à partir de sa propre page, asseyez-vous et profitez-en. Les sauts auront lieu au moment voulu. Un message apparaîtra sur la vidéo en plein écran lorsque vous déplacerez le curseur, vous indiquant quels filtres sont activés.

Comment synchroniser les sauts

  1. Si la vidéo correspond à la capture d'écran dans l'onglet Sync après avoir chargé le fichier de filtre, tout est synchronisé et vous avez terminé cette phase. Revenez au début de la vidéo et commencez à la regarder.
  2. Cependant si la vidéo provient d'une source différente de celle utilisée pour créer le fichier de synchronisation, il peut y avoir une incompatibilité. Utilisez les boutons fléchés pour parcourir la vidéo jusqu'à ce que l'écran corresponde à la capture d'écran ou à la description (image par image, si Fin est coché). Le bouton d'avance rapide change la vitesse ou reprend la lecture si elle est en pause. Vous pouvez redimensionner la capture d'écran avec les flèches et la déplacer si vous maintenez également la touche Alt enfoncée. Maintenez la touche Maj enfoncée également pour des corrections fines (cocher Alt et Fine fait la même chose).
  3. Dans les services qui autorisent les captures d'écran, VideoSkip dispose d'un processus automatique pour trouver le moment de la capture d'écran en cliquant sur le bouton Rechercher capt., pour les services qui permettent de prendre des captures d'écran. Assurez-vous que vous êtes à moins d'une seconde du moment en question avant de cliquer dessus.
  4. Cliquez sur le bouton Synchroniser les heures lorsque la vidéo et la capture d'écran correspondent. Cela décalera tous les sauts de la bonne quantité, de sorte qu'ils se produiront au bon moment. Une nouvelle version du fichier de saut sera créée afin que les personnes avec lesquelles vous le partagez n'aient pas à répéter le processus de synchronisation.

Créer vos propres sauts

  • Un fichier de filtre bien construit aura une première entrée avec le timing d'un événement près du début de la vidéo où l'image change rapidement, plus une description de celui-ci sur la ligne en dessous (n'utilisez pas de chiffres dans le description) et une capture d'écran. Cela permet aux téléspectateurs d'utiliser les sauts dans le fichier même s'ils regardent une vidéo obtenue à partir d'une source différente, qui peut avoir un timing différent. Pour prendre une capture d'écran, faites simplement défiler la vidéo jusqu’au timming souhaitée et cliquez sur Prendre capture ; le moment sera ajoutée à la position actuelle du curseur. Si le service n'autorise pas les captures d'écran, un bouton apparaîtra pour que vous puissiez le charger manuellement. Dans ce cas, prenez la capture d'écran avec la méthode intégrée au système d'exploitation ("outil de capture" sous Windows, command-shift-5 sous MacOS), enregistrez-la dans un fichier, puis chargez ce fichier dans VideoSkip à l'aide du nouveau bouton. Si l'écran contient des zones noires, il est préférable de les exclure et de ne capturer que le film proprement dit.
  • Chaque saut se compose d'heures de début et de fin relatives au début de la vidéo, avec une flèche entre elles, et sur la ligne en dessous d'une catégorie plus peut-être une étiquette de gestion (telle que "vidéo" ou "audio") . L'insertion de sauts est aussi simple que de parcourir le point où le saut/muet/vide est censé commencer, en cliquant sur le bouton Insérer l'heure sur une nouvelle ligne, puis sur la Insérer flèche, en faisant défiler jusqu'au point où le saut doit se terminer, en cliquant à nouveau sur le bouton Insérer l'heure, puis en passant à la ligne suivante et en écrivant un contenu, une évaluation numérique et (facultatif) une étiquette de gestion. Si vous n'êtes pas sûr au départ de l'endroit où les sauts doivent être insérés, il y a une "Avance rapide" bouton qui diffuse la vidéo à grande vitesse. Cliquez à nouveau dessus pour arrêter. Le bouton à sa gauche arrête également la vidéo. Il existe également un bouton Insérer silence qui insère un mot muet rétroactif pour le blasphème ; cliquez dessus dès que vous entendez le mot pendant la lecture normale de la vidéo.
  • Les libellés de contenu ne sont pas sensibles à la casse. Ils se composent de n'importe quel mot (en fait, les trois premières lettres suffisent) des catégories IMDB (anglais), qui sont affichées à côté des cases à cocher, telles que "sex", "nudity", "gore", "drugs", etc. Les étiquettes de traitement sont les mots suivants : "audio", "sound", "video", "image", "blur", "fast" (ou les trois premières lettres de chacun, y compris "img").
  • Les mots-clés "audio" ou "sound" entraîneront la mise en sourdine du son du dans l'intervalle donné pendant que la vidéo est encore affichée, plutôt que de sauter complètement la section, ce qui peut être utile pour supprimer les blasphèmes. Les mots-clés "video" et "image" font disparaître l'image pendant que le son continue de jouer, ce qui peut être utile pour les cas de nudité, etc. "blur" rend l'image floue à la place. "fast" fera passer le film en avance rapide plutôt que de sauter à la fin de l'intervalle marqué. Aucun mot-clé de gestion signifie que la section sera complètement ignorée.
  • Enfin, l'ajout d'un nombre de 1 (léger) à 3 (sévère) permettra aux utilisateurs de définir le niveau de filtrage de manière appropriée. Un bon standard est le suivant : 1 = problématique pour les enfants, 2 = problématique pour les adolescents, 3 = problématique pour les adultes. Il existe une rubrique détaillée sur ce que cela peut signifier sur Videoskip Exchange. Vous pouvez ajouter un texte explicatif entre parenthèses, ce qui ne déclenchera pas les filtres.
  • Tout est modifiable dans la boîte. Utilisez-le pour affiner les horaires. Par exemple, les mots à couper prennent généralement beaucoup moins d'une seconde. Vous pouvez revoir chaque heure en la sélectionnant dans la liste et en cliquant sur Aller à l'heure. Si l'heure est sélectionnée, les flèches décalent l'heure et parcourent la vidéo. Les boutons "Insérer" placent toujours les choses à la position actuelle du curseur.
  • Assurez-vous qu'il y a une ligne vide entre les sauts, comprenant une ligne avec les heures et une ligne avec des étiquettes en dessous.
  • Le bouton Enregistrer enregistrera votre liste, ainsi que la capture d'écran, au format .skp dans votre dossier Téléchargements par défaut. Vous devrez le déplacer de là vers le dossier où vous le souhaitez. Ce format est basé sur du texte, vous pouvez donc afficher le fichier dans n'importe quel éditeur de texte.
  • Le bouton VideoSkip Exchange dans l'onglet Charger ouvrira une page Web où vous pourrez partager votre fichier ignoré avec d'autres utilisateurs de VideoSkip.

Saut d'échantillon, ce qui entraînera la disparition de l'écran à partir de 14 minutes, 8,27 secondes depuis le début du film jusqu'à 14 minutes, 14 secondes, pendant que le son est toujours lu, si "Sexe, Nudité" est défini sur le niveau Haut. Il contient des explications supplémentaires qui n'affecteront pas le filtre:>

0:14:08.27 --> 0:14:14
nude image 1 (homme, de dos)

Autres versions

Il existe également une application Web autonome, qui fonctionne avec les vidéos que vous avez enregistrées sur votre ordinateur. Téléchargez-le sur https://videoskip.org/app pour un ordinateur, ou ici pour un appareil mobile : https://videoskip.org/mobile Les fichiers de saut peuvent être partagés sur le VideoSkip Exchange officiel, ou comme vous le souhaitez. L'échange se fait sur https://videoskip.org/exchange
